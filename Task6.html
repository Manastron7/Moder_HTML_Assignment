Difference between HTML AND HTML5

HTML HTML5
HTML requires a full Document Type Definition (DTD) declaration,
like `< ! DOCTYPE HTML PUBLIC "-//W3C//DTD HTML 4.01//EN">`. 		HTML5 uses a simplified, shorter `< ! DOCTYPE html>` declaration.
HTML lacks specific structural elements like `< header >`, `< nav>`,
`< article>`, and `< section`. 		HTML5 introduces these new semantic elements to provide a
more structured way to define content sections.
HTML relies on third-party plugins like Flash for multimedia content. HTML5 includes native multimedia support with the `< audio >` and
`< video >` elements, reducing the need for external plugins.
