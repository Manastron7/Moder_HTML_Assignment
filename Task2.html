HTML Entities

HTML entities are codes used to represent characters that have special significance in HTML or are not easily typed on a keyboard. These codes begin with an ampersand (&) and end with a semicolon (;). HTML entities are mainly employed to display characters like <, >, &, as well as special characters such as accented letters or symbols in an HTML document. They prevent these characters from being misinterpreted as part of the HTML code and ensure they are displayed correctly in web browsers.

Example of commonly used HTML entities

  1. ' & l t ; ' or ' & # 60 ; 'Represents the less-than symbol '<'
  2. ' & copy ; ' or ' & # 169; ' represents the copyright symbol ©.
  3. ' & amp ; ' or ' & # 38 ; ' : Represents the ampersand &.
  4. ' & euro ; ' or ' & # 8364 ' ; represents the Euro currency symbol €.
  5. ' & reg ; ' or ' & # 174 ; ' represents the registered trademark symbol ®.

⚠ There is no space between the characters
